# PrepWise Resume Builder

一个基于 Vue 3 + Vite 的简历编辑与 AI 面试一体化工具，旨在帮助候选人从简历准备到面试实战，实现全链路的 AI 赋能。

当前版本包含四大核心模块：
- `简历编辑`
- `JD分析`
- `面试题库`
- `AI面试`

## 核心能力

### 1) 简历编辑
- **模块化编辑**：涵盖基本信息、教育经历、专业技能、工作经历、项目经历、荣誉奖项、个人简介。
- **模块开关与顺序调整**：支持可见性切换与拖动式上下调整（基础信息固定在首位）。
- **实时完整度统计**：根据已启用模块动态计算简历完整度。
- **自动本地保存**：编辑内容自动持久化到 `localStorage`，支持手动“保存草稿”。
- **内置多套模板**：提供多种风格模板一键切换。
- **导出能力**：支持导出为高清 PDF、压缩 PDF 及 Markdown。
- **AI 简历优化**：
  - 按模块发起 AI 优化（流式返回）。
  - 输出分为“优化建议”和“优化后内容”。
  - 一键应用优化结果，并支持撤回。

### 2) JD 分析 (JD Analysis)
- **智能解析**：解析目标岗位的 Job Description (JD)，提取核心职责与技术要求。
- **匹配度评分**：自动对比当前简历内容与 JD 要求的匹配度，并给出优化建议。
- **定制化准备**：基于 JD 分析结果为候选人提供针对性的简历修改方向，提高简历通过率。

### 3) 面试题库 (Question Bank)
- **题库生成**：结合简历经历与目标 JD，自动生成定制化的面试题库。
- **卡片式管理**：采用直观的卡片化布局，支持题目分类（技术、业务、软技能等）。
- **自定义笔记与追问**：支持对每道面试题添加个人笔记、模拟追问及考察重点，方便集中复习。
- **状态管理**：支持标记掌握程度（未掌握、熟练、精通），高效追踪准备进度。

### 4) AI 面试 (AI Interview)
- **双模式角色互换**：
  - `你是面试者`（AI 扮演面试官）
  - `你是面试官`（AI 扮演候选人）
- **高逼真模拟场景**：支持流式对话渲染，并搭配专业面试官的语音反馈。
- **面试控制台**：开始、暂停/继续、重置、结束并评分，支持灵活控制面试时长。
- **语音识别与输入**：支持浏览器原生语音输入，快捷键 `Ctrl + I` 快速开关。
- **综合评估报告**：面试结束后自动输出多维度评分（含分项得分与改进建议）。

## 界面概览

本项目采用高度专业、数据密集型的工业风 UI 架构（ui-ux-pro-max），在 JD分析和面试题库中应用了暗黑玻璃拟物风格（Glassmorphism），保障了一致的高级视觉体验。

## 技术栈

- **前端框架**：Vue 3（Composition API）
- **构建工具**：Vite
- **状态管理**：Pinia
- **类型系统**：TypeScript + vue-tsc
- **后端服务 (BaaS)**：Supabase (用于云端面试数据与题库同步)
- **代码质量**：ESLint + Oxlint + Oxfmt
- **导出模块**：html2canvas + jsPDF / html2pdf.js

## 运行环境

- Node.js: `^20.19.0 || >=22.12.0`
- npm: 建议使用最新稳定版

## 快速开始

```bash
npm install
npm run dev
```

默认开发地址：`http://localhost:5173`

## AI 接口配置说明

项目中的各项 AI 功能（简历优化、JD 解析、生成面试题、模拟面试）均支持自定义大语言模型：
1. 打开侧边栏底部的“配置模型”或系统设置。
2. 填写您的 `API URL`（例如兼容 OpenAI 的接口地址）、`API Key` 以及 `Model Name`。
3. 配置会自动保存在本地。

## Docker 一键部署

确保已安装 [Docker](https://www.docker.com/)，在项目根目录执行：

```bash
# 构建并启动
docker compose up --build -d

# 访问应用: http://localhost:3000

# 停止并清理
docker compose down
```

## 注意事项

- **数据隐私**：您的简历数据默认保存在浏览器本地 (`localStorage`)，部分同步功能依赖配置的 Supabase。
- **语音功能限制**：部分浏览器可能限制 Web Speech API 的使用，推荐使用最新版 Chrome 或 Edge 获得最佳语音体验。
